# vintagesquid-dev

This is official the repositoy for the [papisarkany.hu](https://papirsarkany.hu) webshop.

# Prerequisites

- node 22
- pnpm 10

## Getting Started

First, run the development server:

```bash
pnpm i
pnpm run dev
```

Open [http://localhost:4321](http://localhost:4321) with your browser to see the result.
